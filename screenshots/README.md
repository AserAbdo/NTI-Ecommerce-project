Place your app screenshots in this directory with the following naming convention:

## Recommended Image Specs

- **Format:** PNG (preferred) or JPEG
- **Width:** 300-400px (for README display)
- **Aspect Ratio:** Match device aspect ratio (9:16 or 9:19.5)
- **Quality:** Compressed for web (< 500KB each)

## Tips

1. Use device frames for professional look (optional)
2. Capture both light and dark mode if applicable
3. Show real content, not lorem ipsum
4. Hide sensitive information (real user data)
